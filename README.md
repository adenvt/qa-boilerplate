# QA Boilerplate

> Boilerplate API testing, designed for Quality Assurance

## Usage

### 1. Install [SAO](https://github.com/saojs/sao).

```bash
yarn global add sao
# or
npm i -g sao
```

### 2. Run installation

```bash
sao adenvt/qa-boilerplate my-project
```
